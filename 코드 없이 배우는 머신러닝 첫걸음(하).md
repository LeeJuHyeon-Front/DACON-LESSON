# DACON-LESSON
데이콘에서 배운 지식 기반으로 기록 남기는 곳입니다.

1. 머신러닝 모델 학습과 평가를 위한 학습/테스트 데이터 이해

 ▶️ 학습 데이터(Training Data)
 
 ▶️ 테스트 데이터(Test Data)
  
2. 머신러닝 모델 평가를 위한 데이터 구성 세 가지 주요 데이터 파일
   
 ▶️ 학습 데이터(Training Data)
 
 ▶️ 테스트 데이터(Test Data)
 
 ▶️ 제출 양식(Sample Submission)

2.1. 학습 데이터

 ▶️ Train.csv는 AI가 학습하는데 사용하는 데이터로, 분석하며 예측하는 규칙과 패턴탐색
 
2.2. 테스트 데이터

 ▶️ Test.csv는 AI가 학습한 내용을 실제로 적용에 사용, 학습이 끝난 후 이 데이터를 처음 접하게 되며, 학습 과정에서 본 적없는 새로운데이터
 
2.3. 제출 양식(Sample_submission.csv)

 ▶️ AI가 예측한 결과를 제출하는데 사용하는 파일
 
 ▶️ 대회 플랫폼에서 제공하는 파일
 
 ▶️ 테스트 데이터에 대한 예측값을 생성 후, 제출 양식에 저장해 제출하면 대회에서 AI의 성능 평가

3. 학습 데이터, 테스트 데이터 분류 이유

 ▶️ 새로운 상황에서도 작동해야함. AI가 배운 내용 활용 여부 확인
 
 예시 : 색상 맞히기 게임

 ㅁ 초반에 학습데이터(초록, 파랑, 빨강) 학습 -> 이미 본 색상이라 쉽게 적중
 ㅁ 만약 테스 데이터(보라색) 테스트 시 -> 헷갈림
 ㅁ 새로운 데이터 접할 시 잘못된 결과 발생

4. 추가 개념 : 검증 데이터(Validation Data)
   
 ▶️ AI 모델 훈련 과정에서 검증 데이터 사용

 ▶️ 학습 데이터 학습 후 검증 데이터 성능 점검하고 모델 조정하는 과정 거침

ㅇ 데이터셋을 나누는 일반적인 비율(80:10:10 기) 
 ⬛전체 데이터셋
 
   ⬛학습 데이터
   
     ▶️모델이 학습하는데이터
     
     ▶️패턴을 익히는 데이터
   
   ⬛검증 데이터
   
     ▶️모델이 학습 중간에 체크하는 데이터
     
     ▶️과적합을 방지하는 데이터
   
   ⬛테스트 데이터
   
     ▶️모델이 학습이 끝난 후 성능을 평가하는 데이터
     
     ▶️새로운 데이터를 예측하는 능력을 테스트

⭐ 대회 주최측에서 Train.csv, Test.csv 나누어 제공

⭐ 실제 업무
  
  ▶️원본 데이터를 가져와 직접 가공, 학습을 위해 적절한 비율로 Train.csv, Test.csv를 나누는 과정 필요

 5. 지도/비지도 학습의 학습/예측 과정

    ▶️지도 학습(Supervised Learning):정답이 있는 데이터를 학습해 예측하는 방법

    ▶️비지도 학습(Unsupervised Learning):정답 없이 패턴 분석하는 방법

ㅁ지도 학습은 '학습 과정'과 '예측 과정' 두 단계로 이루어짐

🔵 학습 데이터
  - 특징 : AI가 학습하는 입력값
  - 타겟 : AI가 예측해야 하는 정답
    
🔵 학습 진행 과정
  
  1-특징과 타겟이 모두 포함된 학습 데이터를 AI 모델에 입력
  
  2-AI 패턴 학습
  
  3-여러 데이터를 반복 학습하면서 예측 능력 향상
 
🔵 예측 과정  
  
  - AI가 새로운 데이터를 예측하는 단계

🔵 테스트 데이터 구성

  - 특징만 제공

🔵 테스트 진행 과정

  1-새로운 사람의 운동 데이터(특징) 입력
  2-학습된 AI 모델이 패턴을 적용해 예측
  3-예측된 결과 표
